# 组件规格模板（可复用控件）

> 复制本模板到 `02_Components/<ComponentName>.md`，用于写“可复用 + 有契约”的组件规格。

日期：YYYY-MM-DD  
状态：Draft/Ready

## 1) 组件定位

- 组件名：xxx
- 使用场景：哪些页面/模块会用
- 非目标：不做什么（避免组件膨胀）

## 2) Props / 数据契约

- 输入字段：类型/必填/默认值/边界条件
- 输出事件：事件名、payload、触发时机

## 3) 交互规则

- 主交互（click/hover/drag）
- 禁用态/加载态/错误态
- 与其他组件联动的协议（如需）

## 4) 可访问性（A11y）

- ARIA 与键盘交互
- 读屏文案与可替代视图（如适用）

## 5) 导出/复制/序列化（如适用）

- 工作区动态渲染 vs 导出静态化
- 数据一致性规则（同一份数据源）

## 6) 验收标准（AC）与回归

- 最小可验证列表（至少 4 条）
